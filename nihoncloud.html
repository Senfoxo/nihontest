About Our Chat System

Nihon Cloud Storage's chat system is a very complex and continuously expanding feature. It allows users to communicate with each other online through the cloud storage service, enhancing collaboration and interaction among users.

Upload Scripts

You can upload Lua scripts for Nihon through the cloud. These scripts are synchronized with the executor, allowing you to use the new "Send to Socket" and "Execute" functions to run scripts directly from the cloud or send them to the executor via socket.

Follow System

Nihon's follow system allows users to follow other users they like or find interesting. This system is still being expanded to provide more features and better user experience.
